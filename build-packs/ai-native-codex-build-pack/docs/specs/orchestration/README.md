# Orchestration Specs

Canonical orchestration specifications for `ai-native-codex-package-template` live in this directory.

- `DELEGATION-DOCS-META-TEMPLATE.md`: reusable meta template for delegation briefs, documentation-role boundaries, and validation-first doc design.
- `ORCHESTRATION-DELEGATION-ENFORCEMENT-TECH-SPEC.md`: canonical enforcement contract for delegated task briefs, worker returns, docs-role boundaries, and generated agent README requirements.
- `ORCHESTRATION-RULES-ENFORCEMENT-OPTIONS-TECH-SPEC.md`: enforcement layers, validator targets, blocking outcomes, and minimum rollout order.
- `ORCHESTRATOR-AUTONOMY-LOOP-TECH-SPEC.md`: autonomous orchestration loop policy, deterministic task selection, approval handling, and escalation rules.
- `DELEGATED-AGENT-COMMUNICATION-HANDOFF-PROTOCOL-TECH-SPEC.md`: orchestrator-to-worker handoff protocol, canonical worker result schema, and monitoring expectations.
- `ORCHESTRATION-PACK-MODERNIZATION-IMPLEMENTATION-TECH-SPEC.md`: implementation roadmap for machine-readable contracts, templates, validators, and orchestration runtime integration.
- `ORCHESTRATION-PACK-IMPLEMENTATION-EXECUTION-PLAN.md`: sequential micro-task implementation plan for building the modernization spec into the template pack.
