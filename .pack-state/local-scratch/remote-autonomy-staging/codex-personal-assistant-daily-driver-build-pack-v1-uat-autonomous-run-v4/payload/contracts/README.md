# Contracts

This build-pack keeps its runtime assistant baseline in machine-readable contracts.

Current contracts:

- `assistant-profile.json`
- `operator-profile.json`
- `partnership-policy.json`
- `context-routing.json`
- `memory-policy.json`
- `skill-catalog.json`
- `project-objective.json`

Reusable behavior belongs in the source template. Operator-specific values and
runtime refinements belong here.
